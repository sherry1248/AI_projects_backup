핵심 메뉴

상품 등록, 재고 조회, 입고/출고, 이력, 부족 알림으로 화면을 단순화했습니다.

입력 방식

카메라 기반 QR/바코드 인식 또는 상품 ID 수동 입력으로 동작합니다.

PC에서 서버를 실행한 뒤 같은 Wi-Fi의 모바일 브라우저에서 접속합니다.
